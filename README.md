Convirta - Android Currency Converter
Convirta is a mobile application developed for the Android platform that provides real-time currency conversion services. The application connects to an external financial API to fetch up-to-date exchange rates, ensuring accuracy for global currency calculations.

Technical Features
Live Data Integration: Uses the ExchangeRate-API to retrieve current market values.
Dynamic Selection: Allows users to select both source and target currencies through interactive spinners.
Multithreading: Implementation of background threads for network requests to maintain UI responsiveness and performance.
Responsive Design: Built with a vertical LinearLayout to ensure compatibility across various screen sizes.
Visual Feedback: Includes automated animations for the application title and error handling via Toast notifications.
Technologies Used
Programming Language: Java
Environment: Android Studio
Interface Definition: XML Layouts
Data Format: JSON
Protocol: HTTP/REST
Installation and Configuration
Clone the repository to your local machine.
Open the project directory using Android Studio.
Ensure you have a valid API Key from exchangerate-api.com.
Locate the API_KEY variable in the MainActivity.java file and insert your key.
Compile and run the application on an Android Virtual Device or a physical smartphone.
Project Structure
MainActivity.java: Contains the core logic for event handling, network communication, and JSON parsing.
activity_main.xml: Defines the visual structure including inputs, selection menus, and result displays.
AndroidManifest.xml: Configures application permissions including necessary internet access.
